# Atividade: Widgets Stateful e Stateless

A atividade conta com uma série de 04 exercícios para fixação e desenvolvimento de lógica dentro do desenvolvimento mobile com flutter.

- [x] Contador com + e -;
- [ ] Dark/light mode;
- [ ] Lista de Tarefas;
- [ ] Botão de curtir com contador.


Obs.: Devido a defasagem de hardware do desenvolvedor, foi utilizado o site [Dartpad](https://dartpad.dev/) para criar os arquivos e lógicas. Para salvar no github foi copiado e colado o que foi desenvolvido num arquivo com extensão ".dart", e por isso a estrutura de pastas está num formato diferente do que se encontra num projeto de flutter iniciado na máquina.
